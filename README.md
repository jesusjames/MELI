## Desarrollo prueba MELI frontend

Frontend:
- JavaScript
- React
- Redux
- Redux-saga
- Redux-toolkit
- HTML
- SASS
- Styled-components

Backend For Frontend:
- Node Js
- Expresss

## Notas

Clone el repositorio

Ejecutar dentro de la carpeta backend:
```
npm install
npm start
```
El puerto de escucha para el backend es el :4000

Ejecutar dentro de la carpeta frontend:
```
npm install
npm start
```
